# Product

## Register

product

## Users

Lucid Linux is for desktop Linux users, VM testers, builders, and curious technical users who want a polished Debian-based system that feels premium without hiding its Linux foundations. They are usually evaluating the live ISO, installing in a VM, or setting up a fresh desktop where first impressions matter.

## Product Purpose

Lucid Linux provides a bootable and installable Debian-based desktop with a refined KDE experience and an integrated local assistant layer. Success means the system boots reliably, presents itself consistently as Lucid Linux, offers graphical installation, and introduces Lucid Agent without blocking normal desktop use.

## Brand Personality

Calm, glassy, capable. Lucid should feel close to the familiar polish of macOS while remaining original, transparent about its Debian base, and confident as its own Linux distribution.

## Anti-references

Do not copy Apple assets, logos, wallpapers, icons, or exact proprietary UI. Avoid cluttered Linux desktops with random launchers, noisy effects, mismatched icons, default Debian installer branding, and AI-looking decorative gradients that do not serve the interface.

## Design Principles

1. Make the first boot feel intentional.
2. Borrow familiar desktop rhythms, but keep every asset and mark Lucid-owned.
3. Prefer stable KDE-native configuration over fragile visual tricks.
4. Keep the agent helpful and optional, never a blocker.
5. Preserve truth: Lucid is Debian-based and Hermes-powered where applicable.

## Accessibility & Inclusion

Target readable contrast, keyboard-accessible launchers and login flows, reduced visual clutter, and no required animation for understanding. Do not require an API key or network access before the desktop is usable.
