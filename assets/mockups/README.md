## Design Guidelines

Mockups should:
- Use the Lucid Glass color palette (see assets/wallpapers/README.md)
- Show dark theme by default
- Demonstrate the glass blur/translucency effects
- Be created in a tool that exports to PNG or SVG (Figma, Inkscape, GIMP)
- Not copy macOS, Windows 11, or other trademarked UI designs

## How to Create a Mockup

1. Use a design tool (Figma, Inkscape, GIMP, Penpot)
2. Reference the Lucid Glass color palette
3. Save as PNG (for screenshots) or SVG (for vector assets)
4. Name files descriptively: `feature-state.extension` (e.g., `login-screen-v2.png`)
5. Add a comment here describing what the mockup shows

## Penpot (Recommended)

[Penpot](https://penpot.app) is a free, open-source design tool that works in the browser. It's a great fit for LucidOS as it's Linux-friendly and doesn't require proprietary software.
