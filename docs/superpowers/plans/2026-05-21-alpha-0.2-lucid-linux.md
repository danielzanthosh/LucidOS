# Lucid Linux Alpha 0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build Lucid Linux Alpha 0.2: a fully rebranded Debian-based live ISO with macOS-inspired KDE styling, custom assets, graphical install branding, and Hermes Agent preinstalled as Lucid Agent with first-login onboarding.

**Architecture:** Keep Debian live-build as the ISO pipeline and layer Lucid branding through includes, hooks, bootloader overrides, and skel defaults. Preserve upstream Debian and Hermes truth in compatibility/license metadata while making the user-facing session consistently Lucid Linux.

**Tech Stack:** Debian live-build, KDE Plasma 5, SDDM QML, Calamares, Bash hooks/scripts, XDG autostart, Konsole profiles, SVG assets, Hermes Agent.

---

```bash
#!/usr/bin/env bash
set -euo pipefail

ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
cd "${ROOT}"

fail() {
  echo "[FAIL] $*" >&2
  exit 1
}

require_file() {
  [[ -f "$1" ]] || fail "Missing required file: $1"
}

require_text() {
  local pattern="$1"
  local file="$2"
  grep -Fq "$pattern" "$file" || fail "Expected '$pattern' in $file"
}

forbidden_visible=(
  "Install Debian"
  "LucidOS Alpha 0.1"
  "LucidOS Build System"
)

for text in "${forbidden_visible[@]}"; do
  if grep -RIn --exclude-dir=.git --exclude='2026-05-21-alpha-0.2-lucid-linux-design.md' "$text" .; then
    fail "Found old visible branding: $text"
  fi
done

require_file "live-build/config/includes.chroot/usr/share/backgrounds/lucid-wallpaper.svg"
require_file "live-build/config/includes.chroot/usr/share/icons/hicolor/scalable/apps/lucid-logo.svg"

require_text 'ID=lucid' "live-build/config/hooks/normal/0100-lucidos-defaults.hook.chroot"
require_text 'ID_LIKE=debian' "live-build/config/hooks/normal/0100-lucidos-defaults.hook.chroot"

```

Hermes' official Linux installer handles Python, uv, Node.js, ripgrep, and ffmpeg itself. The ISO build only needs Git, curl, and CA certificates available before the installer runs.

- [ ] **Step 2: Add Hermes install hook**

Create the hook with a single install strategy after verifying current Hermes docs:

```bash
#!/bin/bash
set -euo pipefail

log() {
    echo "[Hermes Hook] $(date '+%H:%M:%S') $*"
}

log "Installing Hermes Agent for Lucid Agent"

mkdir -p /var/lib/lucid-agent

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh \
    | HERMES_HOME=/var/lib/lucid-agent/hermes bash -s -- --skip-browser

if ! command -v hermes-agent >/dev/null 2>&1 && ! command -v hermes >/dev/null 2>&1; then
    echo "Hermes Agent installation completed but no hermes command was found" >&2
    exit 1
fi

if command -v hermes >/dev/null 2>&1 && ! command -v hermes-agent >/dev/null 2>&1; then
    ln -sf "$(command -v hermes)" /usr/local/bin/hermes-agent
fi

log "Hermes Agent installed successfully"
```
