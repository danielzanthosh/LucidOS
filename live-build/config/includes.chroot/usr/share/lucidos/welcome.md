# Welcome to Lucid Linux Alpha 0.2

Lucid Linux is a Debian-based live and installable desktop with a Lucid Glass interface and Lucid Agent powered by Hermes.

- Try the system from USB or a VM.
- Use **Install Lucid Linux** for graphical installation.
- Launch **Lucid Agent** for onboarding and automation help.

Base compatibility: Debian Bookworm.
